# datastructures

Java-like object-oriented data structures for Go.

This package give you many generic data structures:
- Collections
   - Array
   - Linked list
   - Read-only wrapper 
   - Concurrent wrapper
- Maps
   - Binary search tree 
   - Hashmap 
   - Linked list map 
   - Read-only wrapper
   - Concurrent wrapper
- Sets
   - Map set (created from a map)
   - Read-only wrapper

You can iterate through all collections by using 
iterators or streams.
